# Contributing to Future Fundi

We welcome contributions to the Future Fundi project! Please follow these guidelines to ensure a smooth development process.

## Getting Started

- Follow the setup instructions in the `README.md` file.
- Create a new branch for your feature or bug fix.

## Code Style

- Follow the existing code style.
- Run linters before committing your changes.

## Pull Requests

- Create a pull request with a clear title and description.
- Ensure that all tests pass before submitting your pull request.
